#Dotfiles

##Files:

- `.DS_Store`: macOS Finder metadata file.
- `.git`: Git metadata directory for this repo.
- `.greeting`: Custom greeting text/config.
- `.mytheme.omp.json`: oh-my-posh theme configuration.
- `.tmux.conf`: tmux configuration.
- `.zshrc`: zsh shell configuration.
- `README.md`: Overview and brief description of this dotfiles repo.
- `grade-calc.sh`: Shell script for calculating grades.
